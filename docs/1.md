# NEST CLI

### Insallation

```sh
npm install -g @nestjs/cli
```

Get help

```sh
nest ---help
```

```sh
nest generate --help
```

### Generate new project

```sh
nest new nest-sample-project
cd nest-sample-project
npm run start:dev
```
